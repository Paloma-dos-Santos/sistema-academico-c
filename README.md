# Sistema Acadêmico com Listas Encadeadas em C

## Descrição
Sistema acadêmico desenvolvido em linguagem C que gerencia disciplinas e alunos utilizando listas encadeadas.

## Funcionalidades
- Criação de disciplinas
- Matrícula de alunos em disciplinas
- Remoção de alunos
- Listagem de disciplinas
- Contagem de alunos por disciplina
- Liberação dinâmica de memória

## Estruturas de dados utilizadas
- Lista encadeada de disciplinas
- Lista encadeada de alunos por disciplina

## Conceitos aplicados
- Estruturas de dados dinâmicas
- Ponteiros
- Alocação dinâmica de memória
- Modularização de funções
- Organização de sistemas

## Linguagem
C
